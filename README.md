# TIC TAC TOE

Typical tic-tac-toe. 2 players required. Board size 3x3 ... 10x10

This app is create by Next.js 
## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser and enjoy
