# CONNECT-4

Connect 4 game
This app is create by Next.js 

With some helps form Claude AI

## Requirements
**node.js** must be installed

**pnpm** is recommened


## Getting Started

Install dependencies

```bash
npm install
# or
pnpm install
```
Start dev server
```bash
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser and enjoy
